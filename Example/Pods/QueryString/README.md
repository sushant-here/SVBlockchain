Model for parsing and appending query strings to URLs.

# Install

```bash
pod 'QueryString'
```

# Usage

```swift
let qs = QueryString(key: "query", value: "stockholm")
let url = qs.append(to: "http://google.com")
print(url) // http://google.com?query=stockholm
```
